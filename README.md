# README

## 操作说明

鼠标控制方向

WASD控制前后左右移动；空格竖直向上移动；左shift竖直向下移动

QE表示摄像机向左/向右旋转

1拉近视角；2拉远视角；R恢复正常视角

将屏幕中心对准物体，按一下左键锁定（物体不能离得太远）；按住4围绕锁定点水平旋转；按3解除锁定
